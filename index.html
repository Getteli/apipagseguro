API - Pagseguro PHP

Scripts e classes para pagamento via API do pagseguro em PHP e JS

API

O Pagseguro possui diversas formas de realizar uma transação. Os mais comuns são: botao fixo levando ao site do pagseguro, um checkout transparente, carrinho de compras e assinatura

Eu criei aqui algumas formas usando o antigo SDK via PHP e com JS para gerar os hash tokens necessários para a compra. A versão mais recente ( página de exemplo - Pagseguro - Checkout transparent (valor fixo) ) pode ser editada e mesclada as outras formas, para ter classes com carrinho ou assinatura por exemplo.

Para mais detalhes e ver os códigos, veja os arquivos no meu GitHub

As versões antigas marcadas como legacy, podem ser ainda usadas tranquilamente, porém é necessário uma atualização em seus códigos (fique a vontade!), apenas não o faço a algum tempo. Aproveite :)

Esta é a classe para realizar o pagamento de um valor fixo, via boleto (algumas classes no parametro seriam as do seu usuário, como algum dado financeiro digitado, email, endereço e etc).

Já esta classe, é parecida com a de pagamento via boleot, mas desta vez por cartão de crédito (veja que existem muitas configurações diferentes).

Aqui temos um exemplo da versão legacy para pagamento com carrinho. Onde podemos recuperar os dados do front com os itens do carrinhos.
